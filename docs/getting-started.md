# Getting Started

MaintainerBench is currently an initial scaffold. It provides the package layout, CLI entrypoint, placeholder commands, templates, and documentation pages that future v0.1 work will build on.

## Local Development

```bash
pnpm install
pnpm test
pnpm build
```

## CLI

```bash
maintainerbench --help
maintainerbench init
maintainerbench lint
maintainerbench eval
maintainerbench report
```

The commands are placeholders in this scaffold. They do not execute agents, modify pull requests, or perform repository risk analysis yet.
